    <h2>Commands: </h2>
    <pre>
        /shop flipkart <Search for phone, laptop,etc>
        /shop flipkart offers dotd
        /shop flipkart offers top
        /shop flipkart offers all
        Add 'price desc' at the end for price high to low. Default is price asc
